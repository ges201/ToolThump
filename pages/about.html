About ToolThump

Simple, client-side tools built with privacy and transparency in mind.

A One-Person Project

Hi, I'm the sole developer behind ToolThump. This project started from a simple desire for a set of straightforward, free online utilities that respect user privacy above all else.

I believe that you shouldn't have to worry about your data when converting a file or generating a password. That's why every tool on this site is built to run entirely in your browser. Nothing is ever sent to a server.

Thank you for stopping by. I hope you find ToolThump useful!

Core Principles

  • Privacy First Your data never leaves your device. Ever.
  • Free and Accessible No ads, no subscriptions, no barriers. Just free tools.
  • Fully Open Source The code is public on GitHub for anyone to inspect or contribute.

Interested in the Code?

This project is open-source. Feel free to browse the repository, open an issue, or contribute!
